Tacos and More

HTML * CSS * JQuery

For my final interactive design project, I was tasked to find a website to redesign and then code. So I went with a local restaurant called Tacos and More.

Github link

features

  • Made use of the JQuery library to make a more interactive website with slide show animations.
  • Conducted user research to determine the direction for the website.
  • Attractive user interface.
  • Used media queries for a more responsive web design.
